# InternRadar

AI-powered internship discovery and opportunity intelligence platform.

## Vision
InternRadar helps students discover internships from government portals, startups, research organizations, and company career pages using AI-powered matching and intelligent tracking.

## Planned Features
- AI-powered internship recommendations
- Resume parsing
- Semantic search
- Government internship aggregation
- Deadline tracking
- Reopen notifications
- Internship tracker dashboard
- AI-generated cover letters

## Tech Stack
### Frontend
- React
- TypeScript
- Tailwind CSS
- React Query

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL

### AI
- OpenAI / Groq APIs
- Embeddings-based semantic search
